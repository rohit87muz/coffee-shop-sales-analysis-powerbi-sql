> **End-to-End Data Analytics Project using MySQL, Power BI, DAX, and Excel**

---

## 📌 Project Overview

This project demonstrates an end-to-end data analytics workflow using MySQL and Power BI. The dashboard transforms raw coffee shop transaction data into meaningful business insights through interactive visualizations and dynamic KPIs.

---

## ❗ Business Problem

The coffee shop operates across multiple store locations and records thousands of sales transactions every day. Although the sales data is available, management lacks a centralized dashboard to monitor business performance, identify trends, and make data-driven decisions efficiently.

---

## 🎯 Project Objective

Develop a single interactive dashboard to:

- Monitor Total Sales, Orders, and Quantity
- Compare Month-over-Month Performance
- Identify Top Products
- Compare Store Performance
- Analyze Customer Purchasing Patterns
- Identify Peak Business Hours
- Support Data-Driven Decision Making
